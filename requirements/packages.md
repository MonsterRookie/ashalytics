# Project Requirements & Components

## Tech Stack
- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Library**: Shadcn UI (Radix Primitives)
- **Icons**: Lucide React
- **AI**: Google Generative AI (Gemini)

## Installed Shadcn UI Components
The following components have been installed and integrated into the application:

### Core Layout & Feedback
- `card` (Card, CardHeader, CardTitle, CardContent, CardFooter)
- `button` (Standard usage)
- `badge` (Status indicators)
- `alert` (Warnings & Error displays)
- `separator` (Visual division)
- `scroll-area` (Custom scrollable containers)

### Form & Interactive
- `input` (Text fields)
- `label` (Form labels)
- `select` (Dropdown menus)
- `collapsible` (Expandable sections)
- `dialog` (Modals & Overlays)
- `tooltip` (Hover information)

## External Packages
- `framer-motion`: For animations.
- `class-variance-authority`: For component variant management.
- `clsx` & `tailwind-merge`: For dynamic class utility.
- `@google/generative-ai`: For backend AI processing.
